interested 还是 interesting？

5分钟快速掌握 -ed 和 -ing 形容词的区别

-ed (人的感受)

"I am interested in this topic."

"She looks bored."

-ing (事物的特性)

"This topic is interesting."

"The lecture was boring."

练习 1: 选择题

The movie was so ___ that we all felt ___.



练习 2: 填空题

This game is ___ (interest), but my sister isn't ___ (interest) in it.


练习 3: 图片题

The student looks ___ because the book seems ___.


📌 规则总结

-ed → 描述人的感受 (I feel ___)

-ing → 描述事物的特性 (This is ___)

✨ 记忆口诀："-ed是人，-ing是物！"
